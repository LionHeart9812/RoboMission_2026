# Welcome to my RoboMission 2026 repository!
Here you can see my code for the comming WRO championship in the RoboMission Junior category. <br>
In RoboMission the task for the robot is to complete a "parcour" in which certain tasks have to be done. <br>
Code is for the _2026 RoboMission Junior_ category <br>
Not all task are done by the robot, due to my incompetence ;)

## Dependencies / Used Modules
- **vscode** - *LEGO Mindstorm EV3 MicroPython + ev3dev-browser (extenstions)*
- **ev3dev** - *ev3 brick software*
- **python** - *v3.13.12  (.venv)*
- **pybricks** - *v1.11*
- **pixy** - *Pixy V2 camera for Lego*

## What can you see here?
- `main.py` -- main launch file  
- `linefollower.py` - file with functions for line-following and recognision
- `artefacts.py` - file with the code for completing the 'artefacts' task <br>
